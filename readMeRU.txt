ЧИТАЙ МЕНЯ
==========

Приложение для управления сотрудниками кинотеатра (интерфейс только на русском языке)
---------------------------------------------------------------

Консольное приложение для сотрудников кинотеатра - Руководство пользователя

Вступление
-----------
В этом руководстве содержатся инструкции по использованию консольного приложения Cinema Employee, разработанного на Kotlin. Приложение предназначено для управления операциями кинотеатра, включая продажу билетов, возврат, отслеживание мест и редактирование информации о фильмах и сеансах.

приступая к работе
------------------
Чтобы запустить приложение, запустите файл Main.kt. Это запустит консольный интерфейс приложения.

Функции и использование
-----------------------
 1. Покупка билета
- Ввод: Выберите вариант покупки билета. Вам будет предложено выбрать фильм и место.
- Вывод: Подтверждение покупки билета с указанием названия фильма, времени сеанса и номера места.

 2. Возврат билета
- Ввод: Выберите опцию возврата билета. Вам нужно будет выбрать билет, который вы хотите вернуть, из списка проданных билетов.
- Вывод: Подтверждение возврата билета, и место будет помечено как доступное.

 3. Отображение доступных и проданных мест
- Ввод: Выберите отображение мест для определенного сеанса. Вам нужно будет выбрать сеанс, для которого вы хотите видеть статус места.
- Вывод: Список мест, показывающий, какие из них доступны, а какие проданы.

 4. Редактирование фильмов и сеансов
- Ввод: Выберите опцию редактирования. Вам будет предложено ввести пароль. После успешной аутентификации вы можете выбрать редактирование сведений о фильме или времени сеанса.
- Вывод: Подтверждение изменений, внесенных в сведения о фильме или сеансе.

 5. Сохранение данных
- Ввод: После внесения изменений или транзакций выберите опцию сохранения данных.
- Вывод: Подтверждение того, что текущее состояние данных сохранено в файлах.

 6. Выход из приложения
- Ввод: Выберите опцию выхода. Приложение запросит подтверждение перед выходом.
- Вывод: Приложение автоматически сохранит текущее состояние и затем закроется.

Пример рабочего процесса
------------------------
1. Запустите приложение: запустите Main.kt.
2. Купите билет:
   - Выберите вариант покупки билета.
   - Выберите фильм и место.
   - Получите подтверждение покупки билета.
3. Верните билет (при необходимости):
   - Выберите опцию возврата билета.
   - Выберите билет для возврата.
   - Получите подтверждение возврата.
4. Отредактируйте информацию о фильме/сеансе (при необходимости):
   - Выберите опцию редактирования.
   - Введите пароль.
   - Внесите желаемые изменения.
5. Сохраните изменения:
   - Выберите опцию сохранить данные, чтобы убедиться, что все изменения сохранены.
6. Выйдите из приложения:
   - Выберите опцию выйти и подтвердите закрытие приложения.

Записи
-------
- Хранение данных: Приложение хранит данные в файлах CSV. Убедитесь, что эти файлы находятся в правильном расположении, указанном в коде.
- Пароль для редактирования: Пароль для редактирования по умолчанию задан в коде. Убедитесь, что вы знаете пароль для доступа к функциям редактирования.

Диагностика
-----------
- Неправильные вводимые данные: Если вы введете неверные данные, приложение предложит вам повторно ввести правильную информацию.
- Данные не сохраняются: Убедитесь, что вы выбрали опцию сохранить данные перед выходом, чтобы сохранить все изменения.
